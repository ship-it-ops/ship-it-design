# SegmentedControl

Pill-styled radio group for filtering the current view. Use when the user
picks one of 2–4 mutually exclusive options that change _how data is
displayed_, not which page they're on.

**Distinct from Tabs**: SegmentedControl filters the current view's data
(`Daily / Weekly / Monthly` rate, `List / Map` view). Tabs change
navigational context.

## Accessibility

- `role="radiogroup"` with one `role="radio"` per option. Radix wires it up.
- Keyboard: Arrow keys move and select, Tab moves out of the group.
- Always pass an `aria-label` describing what's being filtered.
