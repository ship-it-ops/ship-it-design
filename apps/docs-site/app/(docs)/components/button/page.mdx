# Button

five states. **One primary per surface** — everything else defers.

## Accessibility

- Renders a real `<button>`, with all native keyboard and focus semantics.
- Loading state sets `aria-busy="true"` so screen readers announce the change.
- Visible focus ring uses the accent palette; passes WCAG AA contrast in both
  themes.
- Use `asChild` to wrap a routing link without losing button behavior.
