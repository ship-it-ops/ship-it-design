# Spacing tokens

A 4px-based scale defined in `packages/tokens/src/spacing.ts`. Use these for padding,
margin, gap, and any layout space. Avoid one-off arbitrary values.

| Token | Value    | Pixels |
| ----- | -------- | ------ |
| `1`   | 0.25rem  | 4px    |
| `2`   | 0.5rem   | 8px    |
| `3`   | 0.75rem  | 12px   |
| `4`   | 1rem     | 16px   |
| `5`   | 1.25rem  | 20px   |
| `6`   | 1.5rem   | 24px   |
| `8`   | 2rem     | 32px   |
| `10`  | 2.5rem   | 40px   |
| `12`  | 3rem     | 48px   |
| `16`  | 4rem     | 64px   |
| `20`  | 5rem     | 80px   |
| `24`  | 6rem     | 96px   |

## Usage

```tsx
<div className="p-4 gap-2">      {/* padding 16px, gap 8px */}
<div className="mt-6 px-8">       {/* margin-top 24px, padding-x 32px */}
```
