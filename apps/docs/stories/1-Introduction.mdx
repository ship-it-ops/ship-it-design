# Ship-It Design System

Welcome to the Ship-It design system. This Storybook is the **interactive** companion
to the source code in this repo. Every component you see here is the same one consumed
by every Ship-It front-end app.

## How this Storybook is organized

- **Foundations** — Tokens (color, typography, spacing) and design principles.
  Start here if you're new to the system.
- **Components** — One section per component. Each shows live examples and an auto-generated
  props table.
