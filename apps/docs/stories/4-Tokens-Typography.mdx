# Typography tokens

Defined in `packages/tokens/src/typography.ts`:

- `fontFamily` — `sans` (UI) and `mono` (code).
- `fontSize` — `xs` … `5xl` ramp, all expressed in `rem` so users can rescale text.
- `fontWeight` — `regular` (400), `medium` (500), `semibold` (600), `bold` (700).
- `lineHeight` — `tight` (1.2), `normal` (1.5), `relaxed` (1.75).
- `letterSpacing` — `tight`, `normal`, `wide`.

## Usage

```tsx
<h1 className="font-sans text-4xl font-bold leading-tight">Heading</h1>
<p className="text-base leading-normal">Body</p>
<code className="font-mono text-sm">code</code>
```
